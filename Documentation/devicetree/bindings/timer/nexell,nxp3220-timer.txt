Nexell NXP3220 Timer Controller

Required properties:

- compatible : should be "nexell,nxp3220-timer"
- reg : Specifies base physical address and size of the registers.
- interrupts : The interrupt of the first timer
- clocks: phandle to the source clock
- clock-names : must include the following entries:
	clock-names : "timerN"(N= 0,1, n).
- clksource : tiemr channel to uset source clock.
- clkevent : usetiemr channel to uset source clock.

Example:
	timer@208c0000 {
		compatible = "nexell,nxp3220-timer";
		reg = <0x208c0000 0x1000>;
		clocks =  <&cmu CLK_TIMER0_TCLK0>,
		       <&cmu CLK_TIMER0_TCLK1>;
		clock-names = "timer0", "timer1";
		interrupt = <0 106 0>;
		clksource = <0>;
		clkevent = <1>;
	};
